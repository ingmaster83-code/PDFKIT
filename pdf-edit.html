자주 묻는 질문

워터마크 도구와 뭐가 다른가요?

워터마크 추가 도구는 전체 페이지에 동일한 텍스트를 자동으로 반복 삽입하는 용도입니다. PDF 편집기는 텍스트·도형·이미지를 원하는 페이지의 원하는 위치에 자유롭게 하나씩 배치하는 용도로, 서로 다른 목적의 도구입니다.

기존 PDF의 글자를 직접 수정할 수 있나요?

현재는 기존 텍스트 위에 새 텍스트 요소를 덧붙이거나 화이트아웃으로 가리는 방식만 지원합니다. 원본 텍스트를 직접 고치는 기능은 추후 지원 예정입니다.

파일이 서버에 업로드되나요?

아니요. 모든 처리는 브라우저 안에서만 이루어지며 파일은 서버로 전송되지 않습니다.
